## Projetos em Destaque

### Sistema de Atendimento Automatizado (TCC)
Automação completa de atendimento acadêmico com arquitetura baseada em ferramentas open source:

- Chatwoot (gestão de atendimento)
- Evolution API (integração WhatsApp)
- Typebot (interface conversacional)
- n8n (orquestração de workflows)
- NocoDB (gestão de dados)

**Destaques:**
- Transferência inteligente para atendimento humano  
- Integração com base de dados acadêmica  
- Estrutura escalável e de baixo custo
